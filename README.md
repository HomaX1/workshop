# WorkShop

## Technologies, that I used here:
* HTML 4, HTML 5
* SCSS
* Gulp
* Adaptive design
* Cross-browser design
* BEM methodology

## How to open
NodeJS version >= 8.10.0

###In console:
```
npm install
```
You could install gulp if you don't have it:
```
npm install -g gulp
```
Run server:
```
gulp connect
```
###Open in browser link

[http://localhost:1703](http://localhost:1703)

###For production version
```
gulp build
```
